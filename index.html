Introdução

A meta deste jogo é facilitar, de forma interativa, o ensino de HTML para quem nunca teve contato com a tecnologia.

Objetivo

Montar o classico "Hello world!" usando somente HTML, com a opção de se preocupar com identação e fazendo uso de diversas TAGS para fazer-lo.

Estrutura básica

A caixa contém um par de reguas enumeradas e encaixaveis para que seja extendida, caso necessário, oito pares de TAGS, duas opções de texto e seis representantes de espaço para indentação.

Modelo guia

                <html>
                    <head>
                        <title>"nome" da pagina</title>
                    </head>
                    <body>
                        <h1>Titulo do topico</h1>
                        <p>texto a ser exibido</p>
                    </body>
                </html>

Tags

As TAGS disponibilizadas nesta caixa são:

  • html
  • head
  • body
  • title
  • h1
  • p
  • span
  • div

As opções de texto são:

  • Hello
  • world!
  • Acre
  • Navetech
  • My first
  • page!
